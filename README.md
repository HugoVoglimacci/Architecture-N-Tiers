# Architecture-N-Tiers
Mobility project by Hugo Voglimacci

Pour lancer le Back:
springboot : run

Pour lancer le front:

Se placer à la racine du projet puis lancer les commandes suivantes ( nécessite NodeJS ):

npm install

npm start
